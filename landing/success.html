Next steps

  1. Save your license. Drop the attached license.json at ~/.config/kirkforge/packy/license.json on the machine that will run Packy.
  2. Install + start the cognition service. 
    git clone https://github.com/KirkForge/KirkForge-Bots
cd KirkForge-Bots/gargoyle-packy
pip install -r requirements.txt
python src/orchestration/packy_endpoint.py
  3. Start the Discord bot. 
    npm install
cp .env.example .env  # fill in DISCORD_TOKEN, ANTHROPIC_API_KEY
node src/bot/index.js
  4. Register slash commands. One-time per Discord server: ./scripts/register_commands.sh. Then try /mood in your server.
