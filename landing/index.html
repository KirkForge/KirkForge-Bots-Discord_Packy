Structured lore injection: Packy recalls the right war story for the right topic. Keyword + mood scored, never random.

🔥 Controlled chaos layer

ADR-008: Packy can interject unprovoked, lock onto a target, override mood on a command, and sabotage one slash command per day. Off by default. Pro+ only.

🔌 Dual mode: direct or microservice

Direct mode: bot calls Claude/MiniMax itself, no Python. Microservice mode: full cognition pipeline as a separate FastAPI service. Choose your trade-off.

🖥️ Admin endpoint on the cognition service

Local HTTP API: /health, /state, /respond, /admin/license, /admin/update. Bound to 127.0.0.1 by default.

Up and yelling in five minutes

  1. Buy a license. One-time payment, no subscription. Signed JSON file emailed to you instantly.
  2. Drop the license in place. ~/.config/kirkforge/packy/license.json
  3. Install + run. cd gargoyle-packy && pip install -r requirements.txt && python src/orchestration/packy_endpoint.py
  4. Start the bot. npm install && node src/bot/index.js Set BOT_MODE=microservice in your .env.
  5. Register slash commands. ./scripts/register_commands.sh One-time per Discord server. /packy, /mood, /lore, /war.
